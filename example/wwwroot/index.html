PnP Server - example page

This page illustrates two different apps being loaded and used with the PnP Server.

The first app, called "Prime", uses a GET method to send a number, typed by the user, and returns whether this number is prime or not (in which case it also returns the closest prime). It does so by loading a pre-calculated table of primes into the app permanent storage at server start-up - using the AppInit() method.

The second app, called "Modify", uses a POST method to send a form with an image, selected by the user, and a modification to make - either make it sepia, or greyscale (app uses stb_image and stb_image_write to do read/write). Max filesize is 10MB, and allowed formats are JPEG, PNG, and BMP.
